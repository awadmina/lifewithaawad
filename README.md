# LifeWithAwad Website

A static personal brand website for LifeWithAwad. The site is built with plain HTML, CSS, and JavaScript, with separate pages for the main areas of the brand.

## Current Pages

- `index.html` - Home page
- `videos.html` - YouTube playlist and latest uploads
- `beloved.html` - Beloved clothing concept
- `world.html` - Brand pillars
- `work.html` - Creative work placeholders
- `about.html` - About Mina
- `contact.html` - Contact and social links

## Improvement Notes

### Improve

Work page: either add real projects or remove it for now. Placeholder portfolio cards can make the site feel unfinished.

World page: this might be unnecessary as its own page. The "Faith / Life / Visuals / Clothing" idea is good, but it may work better on the homepage instead of a full separate page.

About page: keep it, but add a real photo and a more personal short story. This page should make people trust you.

### Remove Or Combine

Combine World into Home.

Remove Work until you have real work/projects to show.

Keep pages to: Home, Videos, Beloved, About, Contact.

### Most Important Content To Add

A real short bio: who you are, what you make, why faith matters to the brand.

Real social links instead of placeholder Instagram/TikTok links.

A clear contact purpose: "For collaborations, video work, clothing updates, or speaking/project inquiries."

A few real video thumbnails or embedded highlights if YouTube feed is slow.

For Beloved: maybe "Coming soon" email signup or "drop list" instead of just concepts.

## Honest Take

Simplify it a little. Fewer pages, stronger content. The design is already expressive; the next level is making every page feel intentional, not just beautiful.

## Development Notes

- No build step is required.
- Open the HTML files directly or serve the folder with a simple static server.
- Shared styles live in `styles.css`.
- Shared interactions live in `script.js`.
- The YouTube feed only runs on pages with `#videoGrid`, so the homepage stays lighter.
